## Example of Use :
Run the executable in your terminal after compiling:
```
gcc -Wall -Werror -Wextra -pedantic -std=c90 *.c -o monty

./monty bytecode_file.m
```
exemple:
```
$ cat -e bytecodes/00.m
    push 1$
    push 2$
    push 3$
    pall$

$ ./monty bytecodes/00.m
    3
    2
    1
```
